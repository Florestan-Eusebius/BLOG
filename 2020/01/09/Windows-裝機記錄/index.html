Windows 裝機記錄

Posted on 2020-01-09 In Computer Views:

因爲想裝WSL，但更新一直出問題，另外也是覺得現在的電腦太亂了，我決定推倒重來，重裝系統。

開發環境的搭建

裝完Windows，先在Control Panel中進入Programs，點擊Turn Windows features on or off，開啓WSL，然後去應用商店下一個ubuntu18.04，就算搞成了。具體教程見微軟官方文檔。我這裏還嘗試把源換成阿里雲的，教程在此。

下載Vscode。

在WSL中搭建python環境，可以從這裏開始看教程。

Ubuntu中自帶了gcc和g++編譯器，我們只需在vscode中下載相關插件即可。

爲了使代碼好看，可以下載prettier插件。

還需安裝gdb以進行調試，在ubuntu的bash中輸入

即可。

安裝github。可以參考這個教程。在Windows系統下，我也安裝了git。

文檔閱讀、編輯和管理

下載並安裝Office套裝，我安裝了Word，Excel，Powerpoint，Onedrive for business（個人版系統自帶），Visio。Onenote就用系統自帶的版本。

下載Adobe Acrobat以閱讀、編輯pdf文檔。

下載Kindle看電子書。

安裝Typora。

安裝Latex。這裏的基本思路也是安裝在WSL中，從那裏打開vscode進行編輯。先安裝texlive：

完成之後，可以這樣配置。

編輯方面還要關心的是輸入法。微軟拼音輸入法對簡體字還不錯，繁體字甚至沒有按輸入歷史調整出詞位置的功能。因此可以安裝Rime輸入法，用戶可以很自由地定製它，比如我就從這裏搞了幾個詞典下來，讓我的輸入法變得更有文化一些。

上網問題

安裝SSR，做該做的事情。

之後我安裝了Chrome，因爲之前註冊了一些奇奇怪怪的帳號，密碼在裏面存着，我連我註冊了哪些帳號都不記得了，更不用說密碼，萬一哪一天要用，直接在這裏查查就好。

其他小工具

在解壓一個zip文件時遇到問題，我去下載了Bandzip。順便，它還可以解壓rar等多種其他格式。

Microsoft Store中下載歐路詞典，學習用。

Microsoft To Do做計劃。

QQ和微信用於聊天。

TranslucentTB把我的Taskbar變成透明的（什麼惡趣味？）
